# MeterMaid

A premium-style **single JUCE plugin mastering and metering suite** focused on:

- LUFS loudness monitoring
- true peak safety
- stereo correlation and vectorscope
- spectrum analysis
- rule-based mastering guidance
- live LUFS timeline context

## Why MeterMaid exists

Most meters either:
- give raw numbers without enough interpretation, or
- bury the user in too much complexity.

MeterMaid is built to do both:
- **measure accurately**
- **show what happened over time**
- **tell the user what to do next**

## Current status

MeterMaid v0.7 is a **serious engineering baseline**.

It is not yet marketed as fully validated reference-grade release software until the following are complete:
- reference comparison pass
- host verification pass
- realtime/performance verification pass
